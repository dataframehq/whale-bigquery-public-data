# `crypto_ethereum.balances`
`bq-1` | `bigquery-public-data`
This table contains Ether balances of all addresses, updated daily.
Data is exported using https://github.com/medvedev1088/ethereum-etl

## Column details
* [STRING]    `address`
  - Address
* [NUMERIC]   `eth_balance`
  - Ether balance
